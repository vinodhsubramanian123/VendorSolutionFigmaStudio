# Live Mission Console Snapshot
**Approved State: v1 (Initial Freeze)**

## Layout Structure
- **Sidebar**: Standard navigation
- **Internal Mission Sidebar**:
  - Details for the selected UCID (Unified Configuration Identifier)
  - Mission status, Timeline, budget estimates
- **Main Mission View**:
  - Dynamic display based on UCID current step:
    - BoQ Intake, Pre-Intelligence, Solution Design, Vendor Provisioning, Comparison.
  - Snapshot viewer (Locked contracts).
- **Consolidation Hub**:
  - Global view for all active missions.
  - Aggregated metrics, grouped by priority.
