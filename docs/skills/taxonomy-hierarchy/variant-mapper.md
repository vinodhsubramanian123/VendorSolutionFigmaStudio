# Taxonomy: Variant Mapper

**Domain**: Taxonomy & Hierarchy

## Purpose
Maps distinct localized SKUs or specific revisions into a single logical "Variant Group" for pricing and stock comparisons.

## Protocol
- Unifies base units (e.g., BTO vs CTO models of the same chassis) to allow comparative analysis.
- Resolves "Smart Buy" vs "Configured" SKUs to determine which is cheaper for the exact same underlying hardware specs.
- Prevents the search engine from treating identical functional parts as separate architectural options.
