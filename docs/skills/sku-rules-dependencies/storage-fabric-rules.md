# SKU Rules: Storage Fabric Rules

**Domain**: SKU Rules & Dependencies

## Purpose
Validates Direct-Attach and SAN fabric routing topologies.

## Protocol: Storage Fabric & Transceivers
- **Storage Fabric**: Asserts direct-out SAN Fabric C routing topologies for clustered storage configurations.
- **Networking/DAC**: Verifies that optical and copper transceivers have corresponding DAC or fiber cables if not integrated.
